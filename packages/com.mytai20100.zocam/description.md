# ZoCam

Extreme zoom slider for the stock Camera app on jailbroken iPhones.

## Features

- Vertical white slider to zoom from 1x up to 400x
- Bypasses system zoom limits via direct AVCaptureDevice hook
- Works seamlessly in both photo and video mode
- No settings required — always active when Camera app is open
- Minimal UI, zero overhead

## Compatibility

- iOS 15+
- iphoneos-arm64 only
- Requires MobileSubstrate, Substitute, libhooker, or Ellekit
