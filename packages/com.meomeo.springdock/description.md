# SpringDock

Customizable SpringBoard dock with live blur, icon scaling, and gesture support.

## Features

- Dynamic blur intensity control
- Per-icon scale and spacing adjustment
- Swipe gestures to switch app pages
- Supports iOS 15, 16, and 17
- Fully compatible with Sileo and Cydia

## Compatibility

- iOS 15 – 17
- iPhone & iPad
- Requires MobileSubstrate
